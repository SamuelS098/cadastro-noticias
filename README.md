Cadastro de Notícias

Aplicação desenvolvida em **PHP (Laravel)** com **Bootstrap 5** para gerenciamento completo de **categorias e notícias**.  
Este projeto foi criado para demonstrar **experiência prática com o framework Laravel**, incluindo o uso de **rotas, controllers, Blade templates, validação de formulários, upload de imagens e banco de dados MySQL**.


 Funcionalidades

- ✅ CRUD completo de **categorias**
- ✅ CRUD completo de **notícias**
- ✅ Upload e exibição de imagens
- ✅ Associação de notícias a categorias
- ✅ Controle de status (ativo / inativo)
- ✅ Interface moderna e responsiva com **Bootstrap 5**
- ✅ Feedback visual com **alertas e modais animados**

Tecnologias Utilizadas

- **PHP 8+**
- **Laravel 10**
- **Bootstrap 5.3**
- **Blade Template Engine**
- **MySQL / MariaDB**
- **JavaScript ES6**
