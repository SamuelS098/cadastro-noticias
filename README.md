#  Cadastro de Notícias - Laravel

Projeto desenvolvido em **Laravel** para cadastro, edição e exclusão de notícias com upload de imagens e categorias.

##  Como rodar o projeto

Siga os passos abaixo:

### 1️ Clonar o repositório
```bash
git clone https://github.com/seu-usuario/cadastro-noticia.git
```

### 2️ Acessar a pasta do projeto
```bash
cd cadastro-noticia
```

### 3️ Instalar as dependências do Composer
```bash
composer install
```

###  Gerar o link de storage
```bash
php artisan storage:link
```

###  Rodar o servidor
```bash
php artisan serve
```

###  Acessar no navegador
```
http://127.0.0.1:8000
```

##  Tecnologias utilizadas
- Laravel 10+
- PHP 8+
- SQLite
- Blade Templates
